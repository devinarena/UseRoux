Use Roux

Hello and welcome! This is an interactive guide site for using the superior cubing method, Roux.

You can use the buttons above to navigate through each individual step of solving the cube.

The scramble shown in this example is (green front, white top):

U2 B2 D' B R' D F' R U F2 L2 F2 L2 U2 F L2 B' U2 B' L2 U2

First Block

The first step for solving the cube using Roux is to solve a 2x3x1 block on the left side.

This step is complete intuition, look for pairs that are already made or easy to make.

Try and track pieces during inspection so you don't need to try and look ahead, which is very difficult at this point.

For this example, you can easily solve first block with blue on the side and white on the bottom using:

Z' Y' U M' L' U L2 U' L' R' U' R U2 B
